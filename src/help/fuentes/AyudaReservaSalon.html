Ayuda para la reserva en el Salon Habana

Para comprender la ventada de reserva salon habana, lo principal a tener en cuenta seran tres puntos mencionados a continuación
con sus respectivas funcionalidades.
Los punto son la gestion del cliente que será insertado en la base de datos con la informacion obtenida de la gestión del salón
que haya hecho. La apliacion por lo general ya consta de funcionalidades que facilitan la usabilidad de los apartados (como tooltips,
simplificacion de informacion requerida, pantallas emergentes con Los consiguientes fallos, etc...).

En los siguientes puntos detallaremos la funcionalidad de los componentes y como emplearlos adecuadamente :

  • Gestion del cliente

    • Para el apartado de clientes, la funcion a cometer es obtener los datos introducidos por teclado por el usuario
    para que mas tarde cuando se haya realizado la gestión del salón se inserte toda la informacion en la base de datos.

    Para que se puedan introducir datos primero se debera introducir un codigo al dni y si este ya existe se generan los datos
    en los campos que ya son existentes en la base de datos, de lo contrario si no existe dicho dni en la base de datos
    aparecerán en disponibles los campos para que el usuario pueda introducir datos en ellos.

    Todos los campos de este apartado son obligatorios y no permitira realizar la reserva sin la inserción de los mismos

  • Gestion de la reserva

    • El apartado de gestion de los datos que desea realizar la reserva se basa segun el tipo de evento que seleccione,
    segun el evento seleccionado se podrán modificar algunos campos y otros no. Para todos sera obligatorio seleccionar
    una fecha que sea mayor a la fecha actual.

    Todos los campos de este apartado son obligatorios y no permitira realizar la reserva sin la inserción de los mismos

  • Gestion de los botones

    • Para que final mente podamos realizar la reserva, el usuario habra tenido que rellenar todos los campos obliglatorios
    con los formatos indicados. Si es así el usuario podrá realizar la reserva simplemente seleccionando el boton que
    aparece en la interfaz como aceptar, Ademas antes de realizar dicha reserva el programa se asegura de si preguntando
    si estas seguro de realizar la reserva.

    Existen dos botones mas cancelar y aceptar el boton de limpiar permite que el usuario quizas decide comenzar
    de nuevo la reserva y se limpian los datos de los campos. el boton cancelar simplemente te permite volver al menu
    principal sin realizar ninguna reserva.

En esta ventan existe un componente denominado temporizador que su funcion es:

evitar que el usuario tome mas tiempo del necesario y por lo tanto solo permite que usuario gestione una reserva duran 60s
y tras ese tiempo el boton de aceptar aparece como deshabilitado impidendo asi, realizar una reserva.
